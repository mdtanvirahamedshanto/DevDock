# Security Policy

## Supported Versions

Only the latest major version of DevDock is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| v0.1.x  | :white_check_mark: |
| < v0.1  | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within DevDock, please do NOT report it by opening a GitHub issue.

Instead, send an email to **security@devdock.com**. All security vulnerabilities will be promptly addressed.

We take security seriously, especially given the privileged nature of this desktop application (Docker access, File system access, Process execution).
